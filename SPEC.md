# Thai Visa Assistant - MVP Specification

## Project Overview
- **Name:** Thai Visa Assistant
- **Type:** Telegram Bot + Landing Page
- **Core Functionality:** AI-powered Thai visa Q&A bot for expats
- **Target Users:** Expats living in Thailand, tourists, digital nomads

## Features

### Phase 1 (MVP)
1. **Telegram Bot**
   - Start command with welcome message
   - Menu: Visa Types, Extension Info, Elite Visa, Work Permit, Ask Question
   - AI-powered answers to visa questions
   - Quick links to official sources

2. **Landing Page**
   - Hero section with value prop
   - Features overview
   - Call to action: "Start on Telegram"
   - Testimonials/Social proof

### Tech Stack
- Telegram Bot API (node-telegram-bot-api or grammY)
- Simple Express.js backend
- OpenRouter AI for intelligent responses
- Simple HTML/CSS landing page
- Deployment: Vercel (frontend) + Render/Railway (bot)

## Data
- Pre-built Q&A knowledge base for common visa questions
- Fallback to AI for complex queries

## Success Metrics
- 100 users in first week
- 50+ daily conversations
